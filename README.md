**CoDive**는 사용자들이 소규모 조(그룹)를 이루어 알고리즘 문제를 함께 풀고,

그 결과를 기반으로 **개인 및 조 단위 랭킹을 제공**하는 서비스입니다.

혼자 하는 공부가 지루하거나, 함께 성장하고 싶은 사람들을 위해 만들어졌습니다.

# 프로젝트 셋팅 방법

### 1. 의존성 설치ss

```
npm install
```

### 2. 개발 환경 실행

```
#API 서버 실행 (포트 3001)
npm run api

#프론트엔드 개발 서버 실행 (포트 8080)
npm run dev
```
